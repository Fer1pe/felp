# Política de Privacidade

Última atualização: 25/11/2025

Esta Política de Privacidade descreve como coletamos, utilizamos e protegemos os dados pessoais dos usuários do projeto **“Projeto Integrador de API WhatsApp — Integração com Chatbot via n8n”**, doravante denominado “Serviço”.

## 1. Controlador de Dados
Responsável: Felipe Machado  
Contato: fel1pe.aguil4r@gmail

## 2. Informações Coletadas
Nosso sistema pode coletar e processar as seguintes informações:

- Dados enviados pelo usuário via WhatsApp (mensagens de texto, números de telefone e interações).
- Informações técnicas de integração, como logs, metadados e histórico de atendimento.
- Dados gerados pelo chatbot automatizado (respostas, fluxos e comandos técnicos).

### Coleta indireta
O projeto não coleta dados sensíveis ou biométricos. Não solicitamos CPF, documentos, endereço físico, dados bancários ou informações financeiras, a menos que o usuário forneça voluntariamente em contexto específico.

## 3. Finalidade da Coleta
Os dados são coletados exclusivamente para:

- Prover respostas automatizadas via chatbot.
- Permitir interação e automação operacional do WhatsApp e n8n.
- Melhorar a experiência de uso através de ajustes técnicos, logs e manutenção.

Não vendemos, alugamos ou comercializamos dados pessoais.

## 4. Base Legal
A coleta e tratamento de dados se baseiam em:
- **Consentimento do usuário** ao iniciar interação.
- **Execução de funcionalidades do serviço** (automação e respostas).
- **Interesse legítimo** para manutenção, auditoria técnica e prevenção de abuso.

## 5. Armazenamento e Retenção
- Dados podem ser mantidos em servidores locais ou de terceiros (n8n, provedores cloud, banco de dados).
- Logs técnicos e interações podem ser armazenados temporariamente apenas para depuração, ajustes e melhoria.
- Dados podem ser removidos mediante solicitação do usuário.

## 6. Compartilhamento com Terceiros
Os dados podem ser compartilhados apenas com serviços necessários para o funcionamento:
- WhatsApp Cloud API ou equivalente
- Plataforma n8n
- Provedores de infraestrutura (como servidores, instâncias ou hospedagem)

Nenhum compartilhamento comercial é realizado.

## 7. Segurança
Aplicamos medidas técnicas e razoáveis para proteger as informações:
- Controle de acesso
- Ambiente isolado
- Remoção de dados não essenciais
- Segurança operacional no backend

## 8. Direitos do Usuário
O usuário pode solicitar:
- Confirmação do tratamento
- Acesso aos dados
- Correção ou exclusão
- Revogação do consentimento
